# Anota(Not started)

> Learn. Practice. Grow.

Anota is a modern e-learning platform designed to make online learning engaging, organized, and accessible. It enables learners to explore courses, watch video lessons, track their learning progress, and improve their skills through quizzes and interactive content.

---

## Live Demo

https://anota-five.vercel.app/

---

## About

Anota provides an intuitive learning experience with a clean user interface and a structured course system. Whether you're learning web development, programming, design, or other technical skills, Anota helps you stay organized and focused throughout your learning journey.

---

## Features

- Browse and explore online courses
- Video-based learning experience
- Structured lessons and chapters
- Interactive quizzes
- Track learning progress
- Search for courses
- Save favorite courses
- Fully responsive design
- Modern and intuitive UI
- Fast and optimized performance
